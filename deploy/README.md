# Deployment (EBAZ4205 / Zynq-7010)

FPGA deployment assets for Tiny-Vector-AI.

- `constraints/` — Vivado XDC pin/timing constraints for the EBAZ4205 board.
- `host_app/` — ARM PS host application controlling the PL accelerator over AXI.

Status: planned. See the Roadmap in the top-level [README](../README.md).
